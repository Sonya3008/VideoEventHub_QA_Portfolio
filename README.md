**О проекте:**

VideoEventHub — учебный (вымышленный) продукт, используемый для демонстрации навыков QA в продуктовой и системной разработке.

Проект представляет собой серверное ПО для сбора, обработки и хранения видеопотоков/аудиопотоков и событий с камер видеонаблюдения (CCTV). Основной фокус — стабильность видеопотока, корректный сбор событий и целостность архива.

Проект создан исключительно для портфолио, без использования данных или материалов реальных компаний.
