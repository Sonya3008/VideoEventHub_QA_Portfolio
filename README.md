**О проекте:**

VideoEventHub — учебный продукт, используемый для демонстрации навыков QA в продуктовой разработке.

Проект представляет собой серверное ПО для сбора, обработки и хранения видеопотоков/аудиопотоков и событий с камер видеонаблюдения (CCTV). Основной фокус — стабильность видеопотока/аудиопотока, корректный сбор событий и целостность архива.

Проект создан для портфолио, без использования данных или материалов реальных компаний.
