**VideoEventHub** — серверное программное обеспечение для централизованного сбора, обработки и хранения видеопотоков и событий с камер видеонаблюдения (CCTV).

Система предназначена для мониторинга объектов, анализа инцидентов и последующего просмотра видеоархива. Продукт принимает видеопотоки от камер, обрабатывает события (движение, потеря сигнала и др.), сохраняет данные в БД и предоставляет доступ к ним через API и клиентское приложение.

**Критически важными для продукта являются:**

- стабильность видеопотока,

- корректный сбор и сохранение событий,

- целостность видеоархива.
