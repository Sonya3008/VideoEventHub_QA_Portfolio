**Чек-лист: видеопоток**

- Камера подключается по RTSP

- Live-видео доступно

- Поток восстанавливается после обрыва с допустимым временем восстановления

- Отображается статус камеры при отсутствии потока

- Аудиопоток синхронизирован с видеопотоком

**Чек-лист: события**

- Событие создаётся при триггере

- Событие сохраняется в БД

- Событие привязано к камере и времени

- Отсутствие события логируется

**Чек-лист: архив**

- Видео сохраняется в архив

- Архив доступен по временным интервалам

- Корректно отображаются разрывы архива
